🆕 Update: Tile Master Elite Pro (v2.0)
The current version of the calculator has been enhanced with advanced features for complex design projects and an improved visualization system.
💎 New Layout Capabilities:
"Carpet" Mode: Automatic calculation of perimeter borders. The program calculates the area of the decorative edging and accurately deducts it from the main field tile volume.
"Centerpiece" (Panno) Mode: Calculation for central artistic inlays. The system centers the decorative element and adjusts the main tile cutting layout around it.
🔍 Interactive Visualization:
Dynamic Zoom Function: An interactive scaling tool for the layout scheme has been added. Now you can examine cutting nodes and complex junctions in detail directly on a smartphone or tablet screen.
📦 Comprehensive Bill of Materials (BOM):
The program now accounts for the full cycle of preparation and finishing works:
Primer Consumption: Calculation of the liquid volume required for substrate preparation.
Waterproofing Layer: Determining the weight of the mastic required for "wet zone" protection.
Installation Hardware: Accurate counting of not just glue and grout, but also Tile Leveling System (TLS) clips and auxiliary compounds.

A specialized tool for construction retailers and flooring experts.
Key Capabilities:
Precision Tile Estimates: Calculates the required quantity of tiles based on the net floor area and box dimensions.
Consumables & Hardware: Automatically estimates the volume of adhesive (glue), grout (using an engineering formula), and the exact number of leveling system clips (TLS).
Layout Versatility: Supports multiple installation patterns, including Classic (Grid), Diagonal, and Offset (Staggered) layouts.
Complex Geometry Support: Accounts for architectural details such as niches, ledges, and columns (both rectangular and circular), with automatic area subtraction.
Visual Layout Mapping: Generates an instant schematic visualization of the tile pattern within the specified room shape.
Smart Wastage Margin: Dynamically adjusts the required waste percentage based on the complexity of the room and the number of corners.
Cost Estimation: Provides a total order cost including all main and auxiliary materials for the final checkout.
